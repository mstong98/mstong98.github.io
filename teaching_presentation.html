HTML Media Attribute

What is html media attribute?

The HTML Media Attribute specifies what media/device the linked document is optimized for. This specific attribute is used to specify that the target URL is designed for special devices, such as iPhone, speech, or print media The HTML Media Attribute can accept several values.

This attribute can be used on these elements:

a
area
link
source
style

Examples of media attribute in code:

1) (A link with a media attribute) a href="att_a_media.asp?output=print“ media="print and (resolution:300dpi)"> Open media attribute page for print. a
2) (Link Example Two different style sheets for two different media types (screen and print)): head>link rel="stylesheet" type="text/css" href="theme.css">link rel="stylesheet" type="text/css" href="print.css" media="print"head (the media attribute is styled using css but is written within HTML as we have been doing ourselves with other attributes and pieces of code).
